# ZX Spectrum Emulator — Phase 2.8 Implementation Plan (DD/FD-prefixed IX/IY)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the DD-prefixed (IX) and FD-prefixed (IY) opcode tables. ~79 documented opcodes across ~18 Op classes. Mostly mechanical adaptations of HL-based ops + signed 8-bit displacement.

**Architecture:** New `IndexReg` enum (IX, IY) parameterizes all DD/FD-prefixed Op classes. Each Op installs at both DD and FD tables. Signed displacement reads from `pc+2` (after prefix at pc+0 + opcode at pc+1). Effective address computed as `(idx.read(cpu) + signedD) and 0xFFFF`. T-states are uniformly "HL counterpart + 4". R increment is +2 (prefix + opcode = 2 M1 cycles). Plus one main-table straggler: `LD SP, HL` (0xF9, 6T).

**Tech Stack:** Kotlin 2.1.10, JDK 21, Gradle 8.10, JUnit 5, AssertJ. No new dependencies.

**Reference spec:** `docs/superpowers/specs/2026-05-02-zx80-m1-phase2-8-ix-iy-design.md`
**Branch:** `opus-4.7`
**Base commit:** spec commit. Phase 2.7 must complete first.

## Universal patterns

- All DD/FD ops: `cpu.bumpR(2)` (prefix + opcode count as 2 M1 cycles).
- PC advance varies per op; spec has the per-op table.
- Signed displacement reading: `mem.read(cpu.pc + N).toByte().toInt()` where N depends on the op (usually pc+2, after prefix and opcode).
- Effective address for `(IX+d)`: `(idx.read(cpu) + signedD) and 0xFFFF`. Wraps mod 64K.
- IX-variants and IY-variants share an Op class parameterized over `IndexReg`. Each installs at both `decoder.dd` and `decoder.fd` via the IxOps fragment.
- T-state cost = base (matches HL counterpart) + 4 (prefix penalty). Per-op tests assert exact T-states.

package ru.alepar.zx80.cpu

/**
 * The two Z80 16-bit index registers IX and IY. Used by all
 * DD/FD-prefixed Op classes via parameterization.
 */
enum class IndexReg(val mnemonic: String) {
    IX("IX"),
    IY("IY");

    fun read(cpu: Cpu): Int = if (this == IX) cpu.ix else cpu.iy

    fun write(cpu: Cpu, value: Int) {
        val v = value and 0xFFFF
        if (this == IX) cpu.ix = v else cpu.iy = v
    }
}

import ru.alepar.zx80.cpu.Cpu
import ru.alepar.zx80.cpu.Memory
import ru.alepar.zx80.op.Op
import ru.alepar.zx80.op.OperandFetcher

/**
 * `LD SP, HL` — copy HL into SP. 6 T-states. Single opcode 0xF9 in
 * the main table. No flag changes. The straggler from Phase 2.5 that
 * lands here alongside its IX/IY siblings.
 */
object LdSpHl : Op {
    override val operandLength = 0
    override val baseCycles = 6

    override fun execute(cpu: Cpu, mem: Memory) {
        cpu.sp = cpu.hl
        cpu.pc = (cpu.pc + 1) and 0xFFFF
        cpu.bumpR()
        cpu.tStates += baseCycles
    }

    override fun mnemonic(operands: OperandFetcher) = "LD SP, HL"
}

import ru.alepar.zx80.cpu.Memory

/**
 * `LD IX/IY, nn` — load 16-bit immediate into index register.
 * 14 T-states. R+=2. PC+=4 (DD/FD prefix + opcode + 2 immediate bytes).
 * No flag changes.
 */
class LdIxImm(private val idx: IndexReg) : Op {
    override val operandLength = 2
    override val baseCycles = 14

    override fun execute(cpu: Cpu, mem: Memory) {
        val nn = mem.readWord(cpu.pc + 2)
        idx.write(cpu, nn)
        cpu.pc = (cpu.pc + 4) and 0xFFFF
        cpu.bumpR(2)
        cpu.tStates += baseCycles
    }

    override fun mnemonic(operands: OperandFetcher) = "LD ${idx.mnemonic}, nn"
}

- [ ] **Task 3: DecIx** — `DecIx(idx)`. Like IncIx but decrements (and wraps from 0x0000 to 0xFFFF). 10T, R+=2, PC+=2, no flags.

- [ ] **Task 4: LdIxFromAddr** — `LdIxFromAddr(idx)`. DD/FD 2A nn nn. Reads address from pc+2..pc+3, then `idx.write(cpu, mem.readWord(addr))`. 20T, R+=2, PC+=4, operandLength=2, no flags.

- [ ] **Task 5: LdAddrFromIx** — `LdAddrFromIx(idx)`. DD/FD 22 nn nn. Reads address, then `mem.writeWord(addr, idx.read(cpu))`. 20T, R+=2, PC+=4, operandLength=2, no flags.

- [ ] **Task 6: AddIxPair** — `AddIxPair(idx, srcBits)`. DD/FD 09/19/29/39. The trickiest of the bunch: src bits encode `00=BC, 01=DE, 10=Self (idx itself), 11=SP`. Validate `srcBits in 0..3` in init. Behavior: `val r = Flags.afterAddWord(idx.read(cpu), readSrc(cpu), cpu.f); idx.write(cpu, r.value); cpu.f = r.newF`. Where `readSrc(cpu) = when (srcBits) { 0 -> cpu.bc; 1 -> cpu.de; 2 -> idx.read(cpu); 3 -> cpu.sp; else -> error("unreachable") }`. 15T, R+=2, PC+=2. **Mnemonic:** `"ADD ${idx.mnemonic}, BC/DE/${idx.mnemonic}/SP"` per srcBits.

- [ ] **Task 7: PushIx** — `PushIx(idx)`. DD/FD E5. `cpu.push(mem, idx.read(cpu))`. 15T, R+=2, PC+=2, no flags.

- [ ] **Task 8: PopIx** — `PopIx(idx)`. DD/FD E1. `idx.write(cpu, cpu.pop(mem))`. 14T, R+=2, PC+=2, no flags.

- [ ] **Task 9: ExSpIx** — `ExSpIx(idx)`. DD/FD E3. Like ExSpHl but with idx instead of HL. 23T, R+=2, PC+=2, no flags. Read both bytes from (SP), write idx's two bytes there, idx gets the read values. Use `mem.readWord(cpu.sp)` and `mem.writeWord`.

- [ ] **Task 10: LdSpFromIx** — `LdSpFromIx(idx)`. DD/FD F9. `cpu.sp = idx.read(cpu)`. 10T, R+=2, PC+=2, no flags.

private fun installPairTouching(d: Decoder) {
    for (idx in IndexReg.entries) {
        val table = if (idx == IndexReg.IX) d.dd else d.fd
        table[0x21] = LdIxImm(idx)
        table[0x22] = LdAddrFromIx(idx)
        table[0x23] = IncIx(idx)
        table[0x2A] = LdIxFromAddr(idx)
        table[0x2B] = DecIx(idx)
        table[0xE1] = PopIx(idx)
        table[0xE3] = ExSpIx(idx)
        table[0xE5] = PushIx(idx)
        table[0xF9] = LdSpFromIx(idx)
        // ADD IX,rr — opcodes 09 (BC), 19 (DE), 29 (Self), 39 (SP)
        for (srcBits in 0..3) {
            val opcode = 0x09 or (srcBits shl 4)
            table[opcode] = AddIxPair(idx, srcBits)
        }
    }
}

import ru.alepar.zx80.cpu.Cpu
import ru.alepar.zx80.cpu.IndexReg
import ru.alepar.zx80.cpu.Memory
import ru.alepar.zx80.op.Op

/**
 * `JP (IX)` / `JP (IY)` — jump to the value of IX / IY. Despite the
 * parens, this is NOT a memory dereference — pc is set to idx itself.
 * 8 T-states. R+=2. No flag changes.
 */
class JpIx(private val idx: IndexReg) : Op {
    override val operandLength = 0
    override val baseCycles = 8

    override fun execute(cpu: Cpu, mem: Memory) {
        cpu.pc = idx.read(cpu)
        cpu.bumpR(2)
        cpu.tStates += baseCycles
    }

    override fun mnemonic(operands: OperandFetcher) = "JP (${idx.mnemonic})"
}
